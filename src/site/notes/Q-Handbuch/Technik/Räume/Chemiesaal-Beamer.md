1. Doppelklick auf "CHEMIE_BEAMER"-Icon am Desktop 
   ![CHS_Beamer_Icon.png|100](/img/user/0%20Bilder/CHS_Beamer_Icon.png)
2. "Epson iProjection"-Fenster öffnet sich
   ![CHS_Epson_iProjeczion.png](/img/user/0%20Bilder/CHS_Epson_iProjeczion.png)
3. auf "Verbinden" klicken
4. Beim Beamer Source auf "LAN" setzen
5. Von Beamer angezeigtes Passwort eingeben
6. Nach der Benutzung App schließen, damit andere User es verwenden können.
