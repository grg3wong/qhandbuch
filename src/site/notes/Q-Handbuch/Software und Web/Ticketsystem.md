Für Meldung von Störungen, Schäden udgl.
1. auf https://grg3hag.webuntis.com/ anmelden
2. auf der Startseite evtl runterscrollen
   ![Ticket_Übersicht.png](/img/user/0%20Bilder/Ticket_%C3%9Cbersicht.png)
3. auf "mehr..." klicken
4. Wenn das Problem noch nicht gemeldet wurde, dann ein neues Ticket erstellen.
   ![Ticket_Liste.png](/img/user/0%20Bilder/Ticket_Liste.png)
5. auf "Neu" klicken
   ![Ticket_erstellen.png](/img/user/0%20Bilder/Ticket_erstellen.png)
6. Problem möglichst genau beschreiben
7. das Ticket wird der verantwortlichen Person weitergeleitet
